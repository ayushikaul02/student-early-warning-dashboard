 🎓 Student Early Warning Dashboard (Excel)

 📌 Overview
This project focuses on identifying students at risk of academic failure using data analytics techniques in Excel.

 🎯 Objective
To support universities in reducing failure rates by analyzing student performance and providing early intervention strategies.

 📊 Key Features
- Analysed attendance, study hours, assignment completion, and exam scores
- Classified students into risk categories (Low, Medium, High)
- Built an interactive dashboard for performance tracking

 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Conditional Formatting
- Charts & Data Visualization
- IF Formulas

💡 Insights
- Identified patterns leading to poor performance
- Suggested interventions like mentoring and attendance monitoring
